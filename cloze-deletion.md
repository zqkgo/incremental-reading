# 完形填空（Cloze Deletion）

## 定义

完形填空（Cloze Deletion）是增量阅读中**将摘录转化为可测试闪卡**的核心技术。它将一个陈述句中的关键词替换为空白，要求学习者从记忆中填入正确答案。

> "完形删除构成了快速阅读和学习技术——增量阅读——的核心。"  
> —— SuperMemo, _Twenty Rules of Formulating Knowledge_

---

## 基本结构

```
原始陈述：
艾宾浩斯遗忘曲线显示，记忆在学习后的24小时内衰退最快。

完形填空卡片：
艾宾浩斯遗忘曲线显示，记忆在学习后的[...小时]内衰退最快。

参考答案：24小时
```

完形填空保留了原始句子的**语境**，这有助于：

- 避免记忆孤立事实（防止"死记硬背"）
- 利用语境线索辅助理解
- 每次复习时重新强化整句知识

---

## 为什么选择完形填空而非问答

SuperMemo 推荐优先使用完形填空（而非纯问答），原因如下：

| 维度     | 完形填空               | 问答（Q&A）            |
| -------- | ---------------------- | ---------------------- |
| 创建速度 | 极快（选中词语即生成） | 需要手动思考并撰写问题 |
| 语境保留 | 完整保留原文语境       | 语境可能丢失           |
| 制作难度 | 低                     | 较高                   |
| 适用场景 | 事实、定义、数据       | 解释性、因果性知识     |
| 记忆质量 | 通常足够好             | 理想情况下更深入       |

**实践建议**：当你阅读到一个适合完形的句子时，立即执行完形删除（Alt+Z），不必担心措辞是否完美——第一次复习时再优化也不迟。

---

## 在 SuperMemo 中的操作

1. 在摘录（Extract）阅读界面中，选中要测试的关键词或短语
2. 按 **Alt+Z**（或点击工具栏的"Cloze"按钮）
3. SuperMemo 自动生成一张完形填空闪卡，关键词被 `[...]` 替换
4. 新闪卡进入 SRS 队列，等待第一次复习

### 批量完形

一个句子可以生成多张完形卡片，每次针对不同的关键词：

```
原句：牛顿在1687年发表了他的运动三大定律。

卡片1：[牛顿]在1687年发表了他的运动三大定律。
卡片2：牛顿在[1687年]发表了他的运动三大定律。
卡片3：牛顿在1687年发表了他的[运动三大定律]。
```

但要注意：每张卡片应测试不同且独立的知识点，不要为同一知识重复创建过多卡片。

---

## 高质量完形填空的要素

### 1. 最小信息原则（Minimum Information Principle）

每张完形卡只测试**一个**事实：

```
❌ 差的做法（测试多个信息）：
[牛顿]在[1687年]发表了他的运动三大定律。

✅ 好的做法（分开测试）：
卡片1：[牛顿]在1687年发表了他的运动三大定律。
卡片2：牛顿在[1687年]发表了他的运动三大定律。
```

### 2. 选择有意义的关键词

应该填空的是**核心事实**，而非语法词：

```
❌ 差的做法（填空不有意义）：
牛顿[在]1687年发表了他的运动三大定律。

✅ 好的做法：
牛顿在1687年发表了他的[运动三大定律]。
```

### 3. 避免歧义

填空后的句子应该只有**一个**正确答案：

```
❌ 有歧义（太宽泛）：
[某位科学家]在1687年发表了他的运动三大定律。
（牛顿？胡克？）

✅ 无歧义：
牛顿在[1687]年发表了他的运动三大定律。
```

### 4. 保留足够语境

空白的位置应让答案可以从记忆中被"钩出"，而不是靠排除法猜测：

```
✅ 有语境：
艾宾浩斯的实验对象主要是他[自己]（不是其他受试者）。

❌ 语境不足（孤立事实）：
问：艾宾浩斯遗忘曲线的实验对象是谁？
```

## 完形填空与知识晶体化

在增量阅读的完整流程中，完形填空是**知识晶体化（Knowledge Crystallization）**的关键一步：

```
原文（液态信息：大量、混乱、难以记忆）
    ↓ Extract
摘录（精炼的片段，但仍是被动阅读）
    ↓ Cloze Deletion
完形填空卡片（固态知识：可主动测试、可长期保留）
```

一旦一段信息被成功转化为完形填空并经过几次 SRS 复习，它就从"读过"变成了真正的"记住"。

---

## 常见错误

- **完形太难**：删掉太多关键词，导致无法从语境判断答案
- **完形太简单**：删掉的词可以仅凭语法猜出，失去测试价值
- **内容未理解就完形**：先理解，再记忆（见[知识公式化原则](./knowledge-formulation.md)）
- **过度完形**：一篇摘录产生几十张卡片，造成复习负担
