# 知识公式化原则（Knowledge Formulation Principles）

## 定义

知识公式化（Knowledge Formulation）是指**将原始信息转化为高质量、可长期记忆的闪卡的方法论**。这是增量阅读中决定学习效果的关键技艺。

Piotr Woźniak 在 1999 年总结了**知识公式化的 20 条规则**，其核心思想是：**记忆的质量由知识的表达方式决定，而非仅由复习次数决定。**

> "掌握知识公式化是使用 SuperMemo 最重要的技能之一。精心设计的问题可以将记忆保留率提高数倍。"

---

## 最核心原则：最小信息原则

### 原则 4：坚持最小信息原则（Minimum Information Principle）

> **每张闪卡只测试一个、且仅一个知识点。**

这是所有规则中最重要的一条，违反它会导致：

- 复习时答不出来，反复失败，间隔被重置
- 即使答出来，也不知道是真正记住了还是靠猜测
- 一个知识点的遗忘会拖累整张卡片

**错误示范**：

```
问：光合作用的原料是什么？产物是什么？
答：原料：CO₂和水；产物：葡萄糖和氧气
（这张卡测试了2个知识点）
```

**正确示范**：

```
卡片1：光合作用的原料是[二氧化碳]和水。
卡片2：光合作用的原料是CO₂和[水]。
卡片3：光合作用的产物是[葡萄糖]和氧气。
卡片4：光合作用的产物是葡萄糖和[氧气]。
```

## 20 条规则概览

### 理解类规则（优先于记忆）

**规则 1：不理解就不要记忆**

- 死记硬背无意义材料是时间的浪费
- 遇到不理解的内容，先通过额外阅读或思考理解它

**规则 2：先学习，再记忆**

- 在创建闪卡之前，确保已经掌握了知识的整体框架
- 零散的闪卡在没有背景知识的情况下很难被记住

**规则 3：打好基础**

- 从基础概念开始，再学习复杂细节
- 复杂知识在没有基础支撑时极难记忆

---

### 简化类规则

**规则 4：坚持最小信息原则**（见上文）

**规则 5：使用简单措辞**

- 用最简单、最直接的语言表达
- 复杂句子 = 更难回忆

**规则 6：使用直接问题**

- 避免绕弯子的问法
- 直接问你想测试的那个事实

**规则 7：避免集合类问题**

集合类问题（列举多个项目的问题）是最常见的错误之一：

```
❌ 差的做法：
问：牛顿三大运动定律是什么？
答：①惯性定律 ②加速度定律 ③作用-反作用定律
（一张卡测试了3个独立知识点）

✅ 好的做法：
卡片1：牛顿第[一]定律描述的是惯性。
卡片2：牛顿第[二]定律描述的是加速度与力的关系。
卡片3：牛顿第[三]定律描述的是作用力与反作用力。
```

**规则 8：避免过长卡片**

- 阅读一张卡片超过 10 秒通常意味着内容太多
- 拆分为更小的单元

---

### 结构优化类规则

**规则 9：使用填字格（Cloze Deletion）**

- 完形填空通常是表达知识最高效的方式
- 比问答更快创建，语境保留更好

**规则 10：使用图像**

- 图像记忆比文字记忆更持久
- 在 SuperMemo 中可以给闪卡添加图片或图表

**规则 11：使用记忆术（Mnemonic）**

- 对难以记忆的孤立事实（如数字、日期），可以创建记忆联想
- 例：用谐音、故事、首字母缩写等技巧

**规则 12：利用图形表示**

- 流程图、表格、脑图等可视化形式
- 对理解结构性知识特别有效

---

### 上下文与关联类规则

**规则 13：避免孤立事实**

- 孤立事实（没有语境的知识）遗忘最快
- 应将事实嵌入更大的知识框架中

```
❌ 孤立事实：
问：艾宾浩斯是哪一年出生的？
答：1850年

✅ 有语境的事实：
艾宾浩斯（1850-1909）发现了遗忘[曲线]，奠定了间隔重复的理论基础。
```

**规则 14：个性化与举例**

- 用自己熟悉的场景举例，让抽象知识具体化
- 个人相关的例子记忆保留率更高

**规则 15：情感化**

- 与情感相关的知识更容易被记住
- 尝试关联知识与个人经历或情感

**规则 16：情景化**

- 提供知识发生的背景（时间、地点、人物）
- 增加知识的"钩子"，让回忆更容易触发

---

### 编辑与维护规则

**规则 17：使用好的引用**

- 记录知识的来源，便于未来深入研究
- 良好的引用系统使知识库更可靠

**规则 18：避免死记硬背**

- 如果某张卡片反复遗忘，不是重复得更多，而是**重新设计卡片**
- 频繁遗忘 = 卡片设计有问题，而非自己记忆力差

**规则 19：优化难以记住的卡片**

- 对"顽固遗忘"的卡片，尝试：添加语境、拆分、换措辞、加图像

**规则 20：使用引用来源和日期**

- 记录每条知识的来源和创建时间
- 便于核实、更新过时知识

---

## 先理解，再记忆：最重要的元原则

以上 20 条规则都服从于一个元原则：

```
理解 → 提炼 → 公式化 → 记忆

（不可跳过前三步直接记忆）
```

在增量阅读中，这个原则体现为：

1. **主题阶段**：深度阅读，建立理解
2. **摘录阶段**：提炼关键信息（已经理解的）
3. **完形填空阶段**：将理解的内容公式化为最小知识单元
4. **复习阶段**：测试并强化记忆
