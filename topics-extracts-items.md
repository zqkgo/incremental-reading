# 主题、摘录与闪卡（Topics, Extracts & Items）

## 概述

在增量阅读体系中，所有学习材料都被分为三种基本元素类型：

| 类型          | 英文    | 作用                 | 测试记忆？     |
| ------------- | ------- | -------------------- | -------------- |
| **主题**      | Topic   | 阅读原始文章         | 否（被动阅读） |
| **摘录**      | Extract | 阅读关键片段         | 否（被动阅读） |
| **条目/闪卡** | Item    | 测试是否记住某知识点 | 是（主动回忆） |

这三种元素构成了知识转化的完整链路：**文章 → 摘录 → 闪卡**。

---

## 主题（Topic）

### 定义

主题是**未经提炼的阅读材料**，通常是一整篇文章、书籍章节或网页。它是增量阅读的起点。

### 特点

- 包含原始、完整的文字内容
- **不直接测试记忆**，仅用于阅读和提取
- 拥有自己的 SRS 复习计划（告诉你何时再次打开继续阅读）
- 可以设置**优先级**和**难度**
- 阅读进度被记录，下次打开时从上次中断处继续

### 类比

主题 ≈ 在 Anki 中创建一张"正面是全文、背面为空"的卡片——你的任务不是回忆，而是阅读并提取。

### 操作流程

```
1. 从浏览器/文件导入文章 → 自动创建为 Topic
2. SRS 调度到期 → Topic 出现在学习队列
3. 阅读文章 → 选中重要段落 → Alt+X 提取
4. 阅读到疲劳/无兴趣 → 按 Next 移到下一篇
5. SRS 再次调度 → 下次从中断处继续
```

## 摘录（Extract）

### 定义

摘录是从主题中**提取的关键片段**，它本身也是一种特殊的主题（读取类型，非测试类型）。摘录比原始文章更简短、更聚焦。

### 特点

- 是主题提取操作的直接产物
- 内容比原始文章更精炼，通常是一句或几句关键话
- 拥有独立的 SRS 复习计划
- 可以从摘录中继续提取（产生"摘录的摘录"，进一步精炼）
- 最终成熟的摘录被转化为**闪卡（Item）**

### 摘录的递归性

摘录可以被多轮精炼：

```
原文（2000 字）
    ↓ Extract
摘录 1（200 字关键段落）
    ↓ Extract again
摘录 2（30 字核心句）
    ↓ Cloze Deletion
闪卡（单个知识点）
```

这种渐进式精炼正是"**增量**"二字的体现：不强求一次到位，而是多次迭代逼近最小信息单元。

---

## 条目/闪卡（Item）

### 定义

条目（Item）是增量阅读的最终产物，是一张**可以测试记忆的闪卡**。它主动要求学习者从记忆中提取答案，而不是被动阅读。

### 特点

- **测试记忆**（区别于主题和摘录的核心特征）
- 通常以**完形填空（Cloze Deletion）**或**问答（Q&A）**形式呈现
- 每张闪卡只包含**一个**知识点（最小信息原则）
- 拥有独立的 SRS 复习计划，间隔随成功复习次数延长

### 两种主要形式

**1. 完形填空（Cloze Deletion）**

```
原句：光合作用将二氧化碳和水转化为葡萄糖和氧气。
闪卡：光合作用将 [...] 和水转化为葡萄糖和氧气。
答案：二氧化碳
```

**2. 问答（Q&A）**

```
问：光合作用的原料是什么？
答：二氧化碳和水
```

## 三者关系图

```
文章来源（网页/PDF/书籍）
        │
        ↓ 导入
    ┌───────────┐
    │   主题     │  ← 阅读，不测试
    │  (Topic)  │
    └─────┬─────┘
          │ Alt+X 提取重要段落
          ↓
    ┌───────────┐
    │   摘录     │  ← 阅读精炼片段，不测试
    │ (Extract) │
    └─────┬─────┘
          │ Alt+Z 生成完形填空
          ↓
    ┌───────────┐
    │  闪卡/条目 │  ← 主动测试记忆
    │  (Item)   │
    └───────────┘
```

---

## 三者的 SRS 调度差异

| 元素 | 调度目的            | 典型间隔                  |
| ---- | ------------------- | ------------------------- |
| 主题 | 继续阅读 / 再次处理 | 较短（几天到几周）        |
| 摘录 | 进一步提炼或转化    | 中等（几天到几个月）      |
| 闪卡 | 测试记忆是否保持    | 从1天开始，成功后指数延长 |

---

## 重要原则

- **主题和摘录**：不要试图一次处理完；看到觉得"可以了"就点 Next
- **闪卡**：力求简单，每张只测一个事实；复杂知识拆分为多张
- **不强迫转化**：不是所有摘录都必须变成闪卡，部分材料保留为主题即可
- **质量优于数量**：一张好的闪卡胜过十张模糊的闪卡
