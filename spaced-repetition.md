# 间隔重复（Spaced Repetition）

## 定义

间隔重复（Spaced Repetition，简称 SRS）是一种基于记忆科学的学习技术：**在即将遗忘某条信息之前，对其进行复习**。每次成功复习后，下次复习的间隔时间会自动延长，从而以最少的复习次数实现最长久的记忆。

间隔重复是增量阅读的核心驱动引擎——它决定了所有阅读材料（主题）和记忆卡片（闪卡）何时被呈现给学习者。

---

## 历史起源

| 时间    | 里程碑                                                                                                |
| ------- | ----------------------------------------------------------------------------------------------------- |
| 1885 年 | 德国心理学家 **赫尔曼·艾宾浩斯（Hermann Ebbinghaus）** 通过实验发现遗忘曲线，奠定了间隔重复的理论基础 |
| 1972 年 | **Sebastian Leitner** 发明莱特纳卡片盒系统（Leitner System），第一次将间隔重复落实为可操作的物理工具  |
| 1985 年 | **Piotr Woźniak** 开发出第一个计算机化间隔重复算法 SM-0，后来发展为 SuperMemo                         |
| 2000 年 | Woźniak 将间隔重复扩展到阅读领域，发明**增量阅读**                                                    |
| 2006 年 | **Anki** 发布，将 SRS 带向普通大众                                                                    |

## 工作原理

### 核心机制

```
初次学习
    ↓ 间隔1（例：1天）
第1次复习 → 成功 → 间隔延长
    ↓ 间隔2（例：4天）
第2次复习 → 成功 → 间隔再延长
    ↓ 间隔3（例：10天）
第3次复习 → 成功 → 间隔继续延长
    ↓ ...
```

- **复习成功**：间隔按一定系数（稳定性因子）延长
- **复习失败（遗忘）**：间隔重置，重新开始短间隔复习
- **目标保留率**：通常设置为 90%~95%，即在每次复习时 9 成以上能成功回忆

### 关键参数

- **可提取性（Retrievability）**：当前时刻能成功回忆某知识的概率（即记忆强度）
- **稳定性（Stability）**：记忆保持一定可提取性所需的时长，每次成功复习后增加
- **最优间隔（Optimal Interval）**：使复习效益最大化的等待时长，由算法动态计算

## SuperMemo 算法演化

SuperMemo 算法持续迭代，每一代都在提升预测精度：

| 版本          | 特点                                                                           |
| ------------- | ------------------------------------------------------------------------------ |
| SM-2（1987）  | 经典算法，Anki 至今仍基于此；使用固定的易难系数（E-Factor）                    |
| SM-5 至 SM-15 | 逐步引入学习者的遗忘曲线建模                                                   |
| SM-17         | 引入双组件记忆模型（可提取性 + 稳定性）                                        |
| SM-18（2019） | 最新版本；针对任意复习历史和复习时机优化间隔；引入遗忘指数（Forgetting Index） |

> SM-2 是最广为人知的版本，因 Anki 而普及；但 SM-18 在理论上更精确。

---

## 遗忘指数（Forgetting Index）

遗忘指数是用户为自己设定的"可接受遗忘率"：

- 默认值通常为 **10%**（即每次复习时允许有 10% 的遗忘）
- 遗忘指数越低，复习越频繁，记忆越牢固，但时间成本越高
- 遗忘指数越高，复习越稀疏，时间效率更高，但记忆更不稳定

增量阅读中，主题（Topic）和闪卡（Item）可以设置不同的遗忘指数，以平衡学习深度与效率。

## 间隔重复 vs. 集中复习（Massed Practice）

|            | 间隔重复           | 集中复习（填鸭式）   |
| ---------- | ------------------ | -------------------- |
| 长期保留率 | 极高               | 极低（考后迅速遗忘） |
| 短期成绩   | 略低于集中复习     | 较高                 |
| 时间效率   | 高（最少复习次数） | 低（重复劳动多）     |
| 适用场景   | 终身知识构建       | 短期应试             |

---

## 在增量阅读中的角色

在增量阅读体系中，间隔重复扮演**调度中心**的角色：

1. 每篇导入的文章（主题）都被赋予一个复习时间表
2. 每个从文章中提取的摘录也有独立的复习时间表
3. 最终生成的闪卡（完形填空/问答）同样按 SRS 调度复习
4. **优先级**与**间隔算法**共同决定每天的学习队列

这意味着：即便你同时管理数千篇文章，SuperMemo 也能保证每篇文章在最需要时自动出现，无需手动追踪。
