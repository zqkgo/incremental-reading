# 遗忘曲线（Forgetting Curve）

## 定义

遗忘曲线（Forgetting Curve）是由德国心理学家 **赫尔曼·艾宾浩斯（Hermann Ebbinghaus）** 于 1885 年通过实验发现的记忆衰退规律：**在没有复习的情况下，新学习的信息会随时间快速流失，衰退速度先快后慢，呈指数下降趋势。**

遗忘曲线是间隔重复算法的理论基础，也是增量阅读中一切调度逻辑的科学依据。

---

## 艾宾浩斯的实验

艾宾浩斯用无意义音节（如 "DAX"、"BUP"）作为实验材料，以**自己**为唯一受试者，测量记忆随时间的衰退情况：

| 学习后时间 | 记忆保留率（约） |
| ---------- | ---------------- |
| 20 分钟后  | 58%              |
| 1 小时后   | 44%              |
| 9 小时后   | 36%              |
| 1 天后     | 33%              |
| 2 天后     | 28%              |
| 6 天后     | 25%              |
| 31 天后    | 21%              |

**关键发现**：

- 遗忘在**学习后数小时内**最为剧烈
- 超过一定时间后，记忆衰退速度明显减缓
- 即使过了很久，也会保留一定的"残余记忆"（重新学习时所需时间更短）

---

## 遗忘曲线的数学模型

艾宾浩斯用以下公式描述记忆保留率：

```
R = e^(-t/S)
```

其中：

- **R**：可提取性（Retrievability），即当前能回忆出的概率
- **t**：距上次学习/复习的时间
- **S**：稳定性（Stability），反映记忆的"耐久度"
- **e**：自然常数（约 2.718）

**直觉理解**：

- S 越大（记忆越稳定），曲线下降越缓慢
- S 很小时（新学知识），曲线陡降，迅速遗忘

---

## 复习如何重置遗忘曲线

每次成功的复习都会**提升记忆稳定性 S**，使下一次的遗忘速度变慢：

```
初次学习
  S = 1 天（1天后约遗忘63%）
        ↓ 在遗忘前复习
第1次复习
  S = 3 天（3天后约遗忘63%）
        ↓ 在遗忘前复习
第2次复习
  S = 10 天
        ↓
第3次复习
  S = 30 天
        ↓
...最终达到 S = 数年
```

这就是间隔重复的核心机制：**每次复习都在"重置"并拉长下一条遗忘曲线的斜率**。

---

## 遗忘曲线与最优复习时机

间隔重复的目标是：在记忆即将跌破某个阈值（如 90% 保留率）时进行复习：

```
记忆
保留
率
100% ┤
 90% ┤----●（第1次复习）
     │     ╲
     │      ╲___●（第2次复习）
     │           ╲________●（第3次复习）
     │                     ╲__________...
  0% └─────────────────────────────────▶ 时间
```

**太早复习**：浪费时间，记忆仍然很清晰  
**太晚复习**：已经忘记，需要重新学习，效率低  
**恰好在临界点复习**：用最少次数达到最长保留

SuperMemo 的算法正是通过建模每个知识点的个性化遗忘曲线，计算最优复习时机。

---

## 间隔效应（Spacing Effect）

遗忘曲线揭示了**间隔效应**的原因：

> 将学习分散在多个时间段（间隔学习），比在同等时间内集中学习（集中学习），产生更持久的记忆。

**机制解释**：

- 集中学习时，每次复习都在遗忘曲线的高位，属于"容易但无效"的练习
- 间隔学习时，每次复习都发生在曲线已明显下降之后，大脑需要更努力地"重新激活"记忆，反而强化了神经联结

---

## 遗忘的好处：遗忘是学习的必要条件

反直觉的是，**一定程度的遗忘对深度学习有益**：

- 当你需要努力从记忆中"挣扎着提取"答案时，记忆强化效果最强（"提取练习效应"）
- 等到记忆有所衰退再复习，比记忆仍清晰时复习效果更好
- 这也是为什么增量阅读不试图"在遗忘之前立即复习"，而是**允许一定程度的遗忘后再复习**

---

## 不同类型知识的遗忘速度

遗忘速度受多种因素影响：

| 因素       | 遗忘更慢                 | 遗忘更快           |
| ---------- | ------------------------ | ------------------ |
| 材料意义性 | 有意义、有逻辑联系的内容 | 无意义、孤立的事实 |
| 情感联系   | 有情感价值的事件         | 情感中性的信息     |
| 联想密度   | 与已有知识大量联结       | 与已有知识无联结   |
| 学习深度   | 深度理解                 | 表面记忆           |
| 知识质量   | 符合最小信息原则的闪卡   | 模糊、复杂的闪卡   |

这也是为什么增量阅读强调：先理解，再记忆；先提炼，再转化为闪卡。

## SuperMemo 中的遗忘曲线可视化

SuperMemo 会为每个用户的每类知识，实时建模和更新遗忘曲线，并显示：

- 当前元素的预期记忆保留率
